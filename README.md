# Растровые алгоритмы растеризации

## Описание проекта

Веб-приложение на React демонстрирует работу 4 базовых алгоритмов растризации:
- Пошаговый алгоритм
- Алгоритм ЦДА (Digital Differential Analyzer)
- Алгоритм Брезенхема для отрезков
- Алгоритм Брезенхема для окружностей

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите приложение:
   ```bash
   npm start
   ```

Приложение откроется на http://localhost:3000

## Система координат

- Центр canvas соответствует точке (0, 0)
- Положительное направление Y направлено **вверх** (математическая система координат)
- Целочисленные координаты привязаны к **центрам квадратов сетки**
- Каждый квадрат сетки = 1x1 единица в мировых координатах
- Масштаб регулируется колесиком мыши (zoom)
- Клик по canvas выбирает ближайшую целочисленную точку

## Пример вычислений: Брезенхем (линия) от (0,0) до (10,5)

```
dx = 10, dy = 5
sx = 1, sy = 1
err = dx - dy = 5

Шаг 1: plot(0,0), e2=10 > -5 → err=0, x=1
Шаг 2: plot(1,0), e2=0 <10 → err=5, y=1
Шаг 3: plot(1,1), e2=10 > -5 → err=0, x=2
... (продолжается аналогично)
Итоговые точки: (0,0),(1,0),(1,1),(2,1),(3,2),(4,2),(4,3),(5,3),(6,4),(7,4),(8,5),(9,5),(10,5)
```

Полные вычисления показаны в коде `src/algorithms/bresenhamLine.js`.

## Производительность

Время измеряется усреднением по 10000 запускам только алгоритма (без отрисовки).

**Типичные результаты** (на Intel i7, Chrome):
| Алгоритм | Отрезок (0,0)-(100,50) | Окружность r=50 |
|----------|-------------------------|-----------------|
| Пошаговый | ~0.15 мс | - |
| ЦДА | ~0.08 мс | - |
| Брезенхем линия | ~0.03 мс | - |
| Брезенхем окружность | - | ~0.02 мс |

Брезенхем самый быстрый (только целые операции).
