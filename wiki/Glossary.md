# Глоссарий

| Термин | Определение |
|--------|------------|
| **Парфюмерный базар** | Торговая площадка для оптовой купли-продажи парфюмерной продукции между фирмами-поставщиками и фирмами-покупателями |
| **Маклер** | Специальный посредник, через которого заключаются все сделки на базаре. Имеет доступ ко всем товарам |
| **Фирма-поставщик** | Оптовая компания, поставляющая парфюмерную продукцию на базар |
| **Фирма-покупатель** | Оптовая компания, приобретающая парфюмерную продукцию через маклеров |
| **Товар** | Парфюмерная продукция, характеризуемая названием, видом, ценой за единицу, поставщиком, сроком годности и количеством |
| **Вид товара** | Категория парфюмерной продукции (духи, туалетная вода, парфюмерная вода и т.д.) |
| **Сделка** | Факт продажи определенного количества товара конкретным маклером конкретной фирме-покупателю |
| **Статистика маклера** | Агрегированные данные о деятельности маклера: общее количество проданных единиц и общая сумма сделок |
| **Консолидация** | Процесс обновления данных о товарах путем вычитания проданных единиц из поставленных и удаления обработанных сделок |
| **Аутентификация** | Процесс проверки подлинности пользователя при входе в систему по логину и паролю |
| **Администратор** | Роль пользователя с полным доступом ко всем функциям системы |
| **SQLite** | Встраиваемая реляционная СУБД, используемая для хранения данных приложения |
| **CRUD** | Базовые операции с данными: Create (создание), Read (чтение), Update (обновление), Delete (удаление) |
