# Спецификация проекта

## Диаграммы (PlantUML)

Все диаграммы хранятся в папке `diagrams/` репозитория.

| Диаграмма | Файл | Описание |
|-----------|------|----------|
| Вариантов использования | use_case.puml | Актёры и функции системы |
| Классов | class.puml | Модули и их связи |
| Деятельности (аутентификация) | activity_login.puml | Процесс входа в систему |
| Деятельности (сделки) | activity_deal.puml | CRUD операции со сделками |
| Последовательности (вход) | sequence_login.puml | Взаимодействие при аутентификации |
| Последовательности (сделка) | sequence_deal.puml | Добавление сделки и обновление статистики |
| Компонентов | component.puml | Структура файлов проекта |
| Развертывания | deployment.puml | Инфраструктура: GitHub, CI/CD, Codecov |

## Как отрисовать диаграммы

Вставьте содержимое .puml файлов в:
- [PlantUML Online](https://www.plantuml.com/plantuml/uml)
- [Draw.io](https://app.diagrams.net) (File -> Import)
