# Функциональные требования

## Актёры
- **Администратор** — полный доступ
- **Маклер** — доступ к своим данным (помечено *)

## Основные функции

### Аутентификация
Вход по логину и паролю, определение роли.

### Управление данными (CRUD)
- Маклеры (фамилия, адрес, год рождения)
- Товары (название, вид, цена, поставщик, срок годности, количество)
- Сделки (дата, товар, количество, маклер, покупатель)

### Аналитические отчеты
1. Продажи товара за период
2. Покупатели товара
3. Вид товара с наибольшим спросом (*)
4. Маклер с максимальным числом сделок
5. Маклеры по фирмам-поставщикам

### Специальные функции
1. Обновление статистики маклера при сделке (*)
2. Консолидация товаров на дату
3. Сделки маклера на дату (*)
