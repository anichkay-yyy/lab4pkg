# Парфюмерный базар

Консольное приложение для управления парфюмерным базаром.

**Команда:** dreamteam | **Группа:** ГР13Б | **Вариант:** 9

## О проекте

Система обеспечивает учет маклеров, товаров и сделок между оптовыми фирмами-поставщиками и фирмами-покупателями на парфюмерном базаре. Реализована аутентификация с разграничением прав (администратор / маклер).

## Технологии

- **Язык:** C (C99)
- **СУБД:** SQLite3
- **Тесты:** CUnit (30 тестов, 7 модулей)
- **CI/CD:** GitHub Actions
- **Покрытие кода:** gcov / Codecov
